# Project Design Instructions

Before any design-related work, inspect `moodboard/` first.

Required preflight:

```sh
./scripts/check-moodboard.sh
```

Then open any moodboard files whose modified time or hash changed before editing UI, visual assets, copy tone, color, typography, layout, motion, or parallax behavior.

Current design direction:

- Neo-renaissance aesthetic
- Gold-black premium futurism
- Manuscript and technical sketch language
- Sacred geometry, orbital diagrams, golden-ratio structure
- Beauty plus utility, human-centered engineering

UI copy remains English unless the user explicitly asks otherwise.
